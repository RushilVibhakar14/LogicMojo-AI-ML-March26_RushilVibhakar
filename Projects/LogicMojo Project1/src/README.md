# End-to-End E-commerce Intelligence System

## Project Overview
This project analyzes an e-commerce dataset to build a Customer 360 Analytics Framework.

The project focuses on:
- Data Cleaning
- Data Merging
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Business Insights and Recommendations

---

## Dataset Used
The project uses multiple e-commerce datasets including:
- customers.csv
- orders.csv
- order_items.csv
- payments.csv
- reviews.csv
- products.csv
- sellers.csv

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Key Analysis Performed
- Customer Analysis
- Revenue Analysis
- Product Analysis
- Seller Analysis
- Review Analysis

---

## Key Insights
- Most customers are one-time buyers.
- Revenue fluctuates monthly.
- Certain product categories contribute higher revenue.
- Longer delivery times reduce customer satisfaction.

---

## Project Outcome
Built a unified master dataset and generated business insights from multi-table e-commerce data.
